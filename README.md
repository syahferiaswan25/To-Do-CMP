To-Do-CMP is a simple task management application built using Kotlin and Compose Multiplatform.
The project demonstrates how to build modern declarative UI with shared business logic across multiple platforms.

Features:
- Add, edit, and delete tasks
- State management with Compose
- Clean and scalable architecture
- Shared UI logic across platforms

Tech Stack:
- Kotlin
- Compose Multiplatform
- Coroutine
- SqlDelight
- MVVM Architecture
